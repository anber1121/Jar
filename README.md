##java基础
Android开发以java语言为基础，所以熟练的掌握java的基本使用和特征是非常重要的。
这部分的内容可以参考《java编程思想第四版》，以及《深入理解java虚拟机》，对大部分知识点了然于胸后，那么对java开发的知识基本了解了。选择进一步开发Java EE或者Android都是不错的方向。
